To download the source code for your website:

  1. Return to the tab in which your website is open.
  2. Right-click on any empty space on the page.
  3. Select "View page source" from the dropdown menu
  4. You will see your source code in a new tab. You can now copy and paste it.

To download the source code for your website:

  1. Return to the tab in which your website is open.
  2. Click Safari in the top left of the menu bar
  3. Click preferences
  4. Navigate to Advanced and tick Show Develop menu in menu bar
  5. Click on the Develop menu
  6. Select "Show Page Source from the dropdown menu.
  7. You will see your source code at the bottom of the screen. You can now copy and paste it.
